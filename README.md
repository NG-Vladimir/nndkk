# График служений

PWA для управления расписанием служений по вторникам и воскресеньям.

## Как запустить

PWA требует загрузки через HTTP(S). Откройте папку через локальный сервер:

```bash
# Вариант 1: через npx (нужен Node.js)
npx serve .

# Вариант 2: через Python
python3 -m http.server 8080
```

Затем откройте в браузере: `http://localhost:3000` (npx serve) или `http://localhost:8080` (Python).

## Установка как приложение

В Chrome/Edge: меню (⋮) → «Установить График служений» / «Установить приложение».

## Функции

- Календарь по месяцам — видны только вторники и воскресенья
- Роли: Ведущий, Бэк-вокал, Фоно, Барабаны, Гитара
- Выбор участников из списка для каждой роли и даты
- Управление списком участников (добавление/удаление)
- Данные хранятся локально в браузере
- Работа офлайн после первой загрузки
